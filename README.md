# decodelabs-project2-classification

## What this project does
A supervised learning project that trains a KNN model to predict 
the species of an iris flower based on its measurements.
The model is trained on 80% of the data and tested on the remaining 20%.

## Dataset
Iris Dataset — 150 samples, 3 species, 4 features:
- Sepal length
- Sepal width
- Petal length
- Petal width

## Libraries used
- scikit-learn
- pandas

## Algorithm
K-Nearest Neighbors (KNN) with n_neighbors=5

## Results
Accuracy: 1.0 (100%) on the test set

## How to run
1. Install dependencies: pip install scikit-learn pandas
2. Run: python classifier.py
